<https://www.tecmint.com/create-multiple-user-accounts-in-linux>

## Commands

- adduser
  `sudo adduser [username]`
- Add user to sudo group
  `sudo usermod -aG sudo [username]`
- change UID
  `sudo usermod -u [UID] [username]`
- change GID
  `sudo usermod -g [GID] [username]`
