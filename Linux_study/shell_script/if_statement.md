# if statement

## operator

- `-n`: not null
- `-z`: null
- `-a`: and
- `-o`: or
- string
  - `=`: same string
  - `!=`: not same string
- number
  - `-eq`: same number
  - `-ne`: not same number
  - `-gt`: greater than
  - `-ge`: greater than or equal to
  - `-lt`: less than
  - `-le`: less than or equal to
- file or directory
  - `-d`: directory exists
  - `-e`: file exists
  - `-nex`: file not exists
